# Application for Automated Time Series Analysis

It is a desktop application for automated structural analysis of time series data.

The application provides a full analytical workflow: from data import and preprocessing to feature extraction, segmentation, clustering, Markov modeling, visualization, and report generation.

## Features

- CSV time series import
- Automatic time column detection
- Basic statistical analysis
- Missing value handling
- Scaling and smoothing
- Sliding-window feature extraction
- Time series segmentation
- Segment clustering
- Markov transition matrix construction
- Data visualization
- PDF report export
- Project saving and loading via SQLite

## Technologies

- Python
- PySide6 / Qt
- Pandas
- NumPy
- SciPy
- scikit-learn
- Matplotlib
- SQLite
